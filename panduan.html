Panduan Penggunaan Aplikasi Ujian

Persyaratan Minimum Perangkat

  • Android: Minimal Android 7
  • Windows: Windows 10 / Windows 11 (64-bit wajib)
  • macOS: Minimal macOS 10.13 (High Sierra)
  • iPhone/iPad: Wajib install Safe Exam Browser dari App Store

Panduan Penggunaan di Windows

Menjalankan Aplikasi

  1. Download aplikasi ujian Windows (.exe) dari halaman utama.
  2. Tidak perlu instalasi — cukup simpan file di Desktop.
  3. Double-click aplikasi untuk membuka ujian.

Cara Keluar Aplikasi

  1. Tekan tombol Keluar di pojok kanan bawah.
  2. Masukkan password keluar:
    sman42025
Jika Windows memblokir aplikasi, klik More Info → Run Anyway.

Panduan Penggunaan di Android

Instalasi

  1. Download file .apk dari halaman utama.
  2. Izinkan instalasi dari "Sumber Tidak Dikenal".
  3. Buka aplikasi dan masukkan Token Ujian.

Cara Keluar Aplikasi

Tekan tombol Back sebanyak 5 kali sampai aplikasi menutup.

Panduan Penggunaan di macOS

Instalasi

  1. Download file Safe Exam Browser (.dmg).
  2. Buka file .dmg dan drag aplikasi Safe Exam Browser ke folder Applications.
  3. Jika aplikasi diblokir, buka:
    System Preferences → Security & Privacy → Open Anyway.
  4. Download dan buka file konfigurasi:
    ExUjian.seb

Cara Keluar Aplikasi

Tekan:
⌘ Command + Q
lalu masukkan password keluar:
sman42025

Panduan Penggunaan di iPhone & iPad

Menjalankan Ujian

  1. Install aplikasi Safe Exam Browser dari App Store.
  2. Buka halaman download dan tap file:
    ExUjian.seb
  3. SEB akan terbuka otomatis dengan konfigurasi ujian.

Cara Keluar Aplikasi

  1. Usap layar dari kiri atas ke bawah.
  2. Pilih Quit Session.
  3. Masukkan password:
    sman42025
Download Safe Exam Browser (App Store)

Cara Memulai Ujian

  1. Buka aplikasi ujian sesuai perangkat.
  2. Masukkan Token Ujian.
  3. Baca instruksi ujian dengan teliti.
  4. Tekan Mulai untuk memulai ujian.
